Лабораторна робота 5

Завдання на лабораторну роботу

  1. В робочій гілці власного репозиторію створити папку для виконання роботи
  2. Розробити динамічну веб-сторінку згідно свого варіанту та підготувати її до розгортання
  3. Склад сторінки: menu, 2-3 sections (including section with dynamic subjects list), footer (with autor's name, surname)
  4. Використати технології html, scss, JS, node JS, express, JSON server, docker
  5. Виконати commit, merge (гілка ready), push
  6. Підготуватися до захисту роботи
