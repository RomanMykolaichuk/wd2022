Лабораторна робота 3

Завдання на лабораторну роботу

  1. В робочій гілці власного репозиторію створити папку для виконання роботи
  2. Розробити сайт (склад, волонтерська допомога, облік тощо) відповідно до встановленої теми
  3. Передбачити вибір об'єктів, збільшення та зменшення їх кількості, проведення простих обчислень (вага, обсяг, вартість тощо), виведення результатів по окремим об'єктам та в цілому
  4. Використати технології html, scss (css), js
  5. Виконати commit, merge (гілка ready), push
  6. Підготуватися до захисту роботи
