Завдання на лабораторну роботу

  1. В робочій гілці власного репозиторію створити папку для виконання роботи
  2. Розробити інформаційний сайт відповідно до встановленої теми
  3. Використати технології html, css (scss)
  4. Виконати commit, merge (гілка ready), push
  5. Підготуватися до захисту роботи
